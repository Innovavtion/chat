# Chat-Fullstack

### Installation

```bash
$ pnpm install
```

### Development

- Create `.env` and transfer the data from `.env.development`

```bash
# Client and Server
$ pnpm dev

# Client
$ pnpm dev:client

# Server
$ pnpm dev:server
```

### Production

- Create `.env` and transfer the data from `.env.example` and add you value

```bash
# Build app
$ pnpm build

# Start static server
$ pnpm start
```
